Documentation. PostgreSQL is a powerful, open source object-relational database system. It has more than 15 years of active development and a proven architecture that has earned it a strong reputation for reliability, data integrity, and correctness. It runs on all major operating systems, including Linux, UNIX (AIX, BSD, HP-UX, macOS, Solaris), and Windows. It is fully ACID compliant, has full support for foreign keys, joins, views, triggers, and stored procedures (in multiple languages).

It includes most SQL:2008 data types, including INTEGER, NUMERIC, BOOLEAN, CHAR, VARCHAR, DATE, INTERVAL , and TIMESTAMP. It also supports storage of binary large objects, including pictures, sounds, or video. It has native programming interfaces for C/C++, Java, .Net, Perl, Python, Ruby, Tcl, ODBC, among others

Documentation. Sequelize.js is an ORM (Object/Relational Mapper) which provides easy access to MySQL, MariaDB, SQLite or PostgreSQL databases by mapping database entries to objects and vice versa. It has very powerful migrations mechanism that can transform existing database schema into a new version. It also provides database synchronization mechanisms that can create database structure by specifying the model structure.

Although Sequelize.js is new library, it have been battle tested. Currently web sites like Shutterstock, Clevertech, Metamarkets and Innofluence have announced that they are using Sequelize.js as ORM of their website.

  • Dashboard

    This module is designed to help managers to take decisions by showing them the most important information that the web app is generating, data like a chart for the financial status, that include the expenses and the increses, also the current tasks, some contacts to make faster chat and little number of projects that you have create.

  • Projects

    This is the main module, to create projects that will have all the information necesarry, to create a new project first you have to make a category, rather be new, in progress, cancelled, complete, etc. So after you create the category you go to All Projects and click on new project enter all the information and click on save, after that the table is going to refresh automatically and you will be able to see, update, and delete.

  • Tasks

    After the creation of a project you can start creating new Tasks for that project or any other, in this module you will have to options to see all of your assigned tasks or all the existing tasks, in this case to create a new one, just click on new task and enter the information, click on save and the table is going to refresh automatically and you will be able to see, update, and delete.

  • Teams

    The team module is important to keep the information of all the employees that work for a project and what tasks they have, so first include a new member to the web app, go to Members and click new Member enter the information of the user and then go to Team Roles to create a new role, this is to give a user the permissions that he can have, for example user can see projects but cant add new team members, after that go to All Teams and click on new team member, choose the member you create and the project that works on, and give that user the role it has.

  • Finance

    In project management it is important to keep the financial status of every entry of the project so in this module it is included an entry category, so first go to Categories and create a new type, and then create a new Entry by going to Finance and create new entry, enter the information with the amount of money and the category of entry, and save it, after that the table is going to refresh automatically and you will be able to see, update, and delete.

  • Notes

    It is always important to take notes of important information that can help us or the team in the future so the notes module will have your personal notes, and also the notes of the projects, to the employees to know when a note is created, everything you do is go to All Notes, create new note and enter the information, after that the table is going to refresh automatically and you will be able to see, update, and delete.

  • Maps

    This module was to look over different locations to know important aspects like where are the project is going to be develop, if there are other locations the company can use, where are the competitions and how can we keep in track of the services the company is going to offer with the creation of the project that and more.

  • Settings

    This module is only to modify your own personal information, this kind of information is the one your employees are going to see, so the user most have the chance to update the information in case it is necessary

    Project Routes
  • /project/list - List all the projects
  • /project/list-tasks - List all the projects and tasks
  • /project/insert - Insert a new project
  • /project/update/:projectId - Update a project with id
  • /project/delete/:projectId - Delete a project with id

    Requeriments:

  1. Install Angular CLI
  2. Install Dependencies, enter to client and in cmd enter the command: npm install

    Steps:

  1. Start the Restfull API, enter to api and in cmd enter the command: npm run start:dev it will run in the port 8000
  2. Start the Angular Client, enter to client and in cmd enter the command: npm start it will run in the port 4200
  3. Enter in a browser to localhost:4200/
